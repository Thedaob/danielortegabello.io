Sobre el proyecto

¿Qué es ImportorDiscord?

ImportorDiscord es un bot desarrollado en Python que automatiza tareas dentro de servidores de Discord utilizando la API oficial de Discord. El sistema está diseñado para simplificar la gestión diaria de comunidades digitales, eliminando trabajo repetitivo y garantizando consistencia en las operaciones del servidor.

Utiliza autenticación mediante token seguro almacenado en variables de entorno, lo que garantiza que las credenciales nunca quedan expuestas en el código fuente. La arquitectura está orientada a la escalabilidad, permitiendo añadir nuevos comandos y módulos sin necesidad de reestructurar el core del sistema.

importor_bot.py 
import discord
from discord.ext import commands
import os

# Configuración con token seguro en .env
intents = discord.Intents.default()
intents.message_content = True
bot = commands.Bot(command_prefix="!", intents=intents)

@bot.event
async def on_ready():
    print(f"Bot activo: {bot.user}")

@bot.command(name="limpiar")
@commands.has_permissions(manage_messages=True)
async def clear(ctx, amount: int):
    await ctx.channel.purge(limit=amount)
    await ctx.send(f"✅ {amount} mensajes eliminados.")

bot.run(os.getenv("DISCORD_TOKEN"))

Funcionalidades

Características principales

Autenticación segura

Token de Discord almacenado en variables de entorno .env, sin exposición en el código fuente ni en repositorios.

Sistema de comandos

Comandos personalizados con prefijos definibles, control de permisos por rol y respuestas dinámicas en tiempo real al canal correspondiente.

Moderación automática

Gestión automática de miembros, limpieza de mensajes en masa, silenciado temporal y registro de acciones de moderación con timestamp.

Procesamiento en tiempo real

Conexión persistente al gateway de Discord con reconexión automática ante interrupciones, garantizando disponibilidad continua del bot.

Gestión de comunidad

Bienvenidas automáticas a nuevos miembros, asignación de roles por defecto y mensajes de confirmación en canales privados de moderación.

Arquitectura extensible

Estructura modular basada en cogs de discord.py, que permite añadir funcionalidades sin afectar el funcionamiento del sistema existente.
