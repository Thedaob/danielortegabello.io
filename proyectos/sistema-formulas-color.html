Sobre el proyecto

¿Qué es este sistema?

El Sistema de Fórmulas de Color es una aplicación web desarrollada a medida para Tecnítex Pinturas, empresa productora de pinturas y recubrimientos. Centraliza todo el catálogo de fórmulas en una base de datos privada MySQL sobre servidor Apache, accesible internamente por el personal autorizado mediante una interfaz web limpia, intuitiva y completamente protegida.

Antes del sistema, las fórmulas vivían en hojas de papel y Excel desincronizados, generando errores costosos en planta. El sistema reemplazó ese proceso por un flujo digital completo: el operario ingresa el nombre del color, obtiene la fórmula exacta con los pigmentos y sus cantidades base, y puede calcular proporciones al instante para cualquier volumen de producción diferente al estándar de 10 litros.

Funcionalidad clave

Calculadora de proporciones

Una de las funcionalidades más solicitadas por el equipo de planta: las fórmulas del catálogo están definidas para 10 litros base. El operario solo ingresa el volumen que necesita preparar y el sistema recalcula automáticamente cada pigmento en proporción exacta, eliminando errores de conversión manual.

Funcionalidades

Qué hace el sistema

Búsqueda exacta por nombre

El operario escribe el nombre del color (ej: Amarillo Oro, Azul Pacífico) o usa los botones de colores frecuentes para autocompletar. Respuesta inmediata desde la BD.

Calculadora de proporciones

Las fórmulas están definidas para 10 L base. El sistema convierte automáticamente cada pigmento al volumen de producción ingresado por el operario, sin cálculo manual.

Colores frecuentes con un clic

Botones de acceso rápido para los colores más consultados del catálogo Tecnítex: Amarillo Oro, Azul Marino, Blanco Ostra, Aguamarina, Blanco Hueso, Azul Pacífico.

Creación de nuevos colores

Formulario protegido para alta de colores con nombre, pigmentos y cantidades. Validación de nombre duplicado y campos obligatorios. Solo usuarios autorizados.

Eliminación con confirmación

Baja segura de colores del catálogo con ventana de confirmación. El sistema previene eliminar colores activos asociados a fórmulas en producción.

Acceso protegido y por roles

Sistema de autenticación con sesiones PHP. Roles diferenciados: operario (solo consulta) y administrador (gestión completa). Base de datos privada en servidor local Apache.
